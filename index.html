Disks:
To load a disk:
  • Select a disk. (this will insert the selected disk into the A: drive).
  • Type: load "*" then Enter.
  • Most programs will run automatically, just type "run" if you got an "ok" after load.
Keyboard mapping:
  • The PC keyboard is automatically mapped to the right TVC keys (it can fail on some browser + OS combinations).
  • The built in joystick is mapped to the cursor keys, fire => TAB.
